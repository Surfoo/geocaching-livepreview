Help

Why this tool?

When you write your description, you need to click to save, and then refresh the description in a second tab in order to get the result. If it's not correct you need to repeat the process. What a waste of time!

Geocaching Live Preview let you write your geocache description in HTML, and see it in real time with the real render from geocaching.com.
This Live Preview is an "expert" mode for people which already know and write their descriptions in HTML. But of course you can learn some basic skills to improve the quality of your descriptions.

How to write the description?

  • First, you need to learn some HTML skills. There are a lot of resources on internet: developer.mozilla.org, tutorialrepublic.com, etc.
  • Optionally, you can also learn some CSS skills: developer.mozilla.org
  • When you are satisfied about your description, copy the HTML code source.
  • On the geocache page, click on the button "</> Source".
  • Paste it in the textarea, and save it.
